<h1 align="center">
  BOOK REST API
</h1>

Find and create books on MongoDB Atlas. <a href="https://documenter.getpostman.com/view/12091966/T1DjkKuD?version=latest">Demo</a>

Made with NestJS And MongoDB.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
